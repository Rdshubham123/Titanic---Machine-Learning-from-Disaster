## Dataset Information

This project uses the **Titanic: Machine Learning from Disaster** dataset
provided by Kaggle.

The dataset contains passenger information such as:
- Passenger class
- Sex
- Age
- Number of siblings/spouses aboard
- Number of parents/children aboard
- Fare paid
- Port of embarkation
